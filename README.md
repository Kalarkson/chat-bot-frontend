# AI Chat Frontend

The client part of the chatbot with support for Markdown, LaTeX, authorization and streaming generation of responses from the local LLM.

## Technology

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide Icons**
- **Marked + KaTeX** (Markdown + LaTeX)
- **Ollama API** (local LLM)
